把握度评分

可验证推理，
非 LLM 自报置信度

三因子公式透明可追问，评委与客户都能验证「为什么敢动 / 为什么不敢动」。

总分 = round(100 × (0.35×数据 + 0.35×手册 + 0.30×推理))
总分分级动作权限
≥ 80🟢 可执行建议执行（人工确认高风险）
60 – 79🟡 需确认请负责人确认
< 60🔴 需升级@值班 / 升级审批

核心创新

答辩必讲三件套

跨信号 Agent 运维推理

处置手册约束下的根因链，不是基础设施变更关联。聚焦 Agent 运行态层。

每步可验证

把握度评分三因子分解，数据·手册·推理均可追问公式，零幻觉容忍。

处置可闭环

建议 + 把握度 + 重试/升级，全程留痕。👍👎 飞轮驱动手册越用越准。
