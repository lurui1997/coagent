技术选型 · 理由
维度选型理由
Web 框架FastAPI原生 async、OpenAPI 自文档、类型约束与 Pydantic 一体
契约校验Pydantic v2AgentEvent / LLMOutput 强类型，零幻觉容忍
LLM 调用httpx async不阻塞事件循环；OpenAI 兼容可换模型
诊断模式ReAct Agent按需调工具，比 batch 更贴近真实运维排查
实时 UISSE + HTMX单向审计流足够；免 SPA 构建链，运维台快速迭代
持久化SQLite单机零运维、审计留痕、黑客松/私有化友好
场景配置JSON SSoTPlaybook / 场景 / 评分规则同源，改配置即改行为
评分引擎确定性公式独立于 LLM 自报置信度，评委与客户可追问
